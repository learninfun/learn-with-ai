

1. 特定順序遍歷二叉樹（Medium）
給定一個二叉樹和一個正整數k，按照以下步驟打印出所有節點：
1. 首先打印以根節點為開始的前k個節點。
2. 接著打印所有由前k個節點的子節點分支出去的節點。
3. 重複步驟2，直到沒有更多節點可以打印。
例如，如果k為2，下圖中二叉樹的節點遍歷順序為：1，2，3，4，5，6。

答案: https://leetcode.com/problems/print-binary-tree-in-order-of-levels/
