另外，防火牆還可以設置一些規則，例如限制特定IP地址或特定端口的訪問，從而使網絡更加安全。例如，如果一個公司僅允許一些特定IP地址訪問其內部網絡，那麼它可以使用防火牆限制除了這些IP地址以外的所有訪問者。
