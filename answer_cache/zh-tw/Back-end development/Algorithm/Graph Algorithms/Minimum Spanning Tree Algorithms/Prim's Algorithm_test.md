

1. 給定以下未經排序的圖形的邊權重，求 Prim's 算法找到的最小生成樹的權重總和：
```
    A --2-- B --1-- C
   /    |       /    |
 8     3     1     5
 /      |     /      |
D --2-- E --4-- F --3-- G
```
答案：11
